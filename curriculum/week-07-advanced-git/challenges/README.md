# Week 7 — Challenges

Two open-ended problems. Unlike the exercises, these have **no single right answer** — the point is to design something that works and to defend your choices.

1. **[Challenge 1 — Automate repo checks with hooks](challenge-01-automate-repo-checks-with-hooks.md)** — build a shareable, team-grade hook setup that enforces real quality gates.
2. **[Challenge 2 — Track down a regression across many commits](challenge-02-track-down-a-regression-with-bisect.md)** — hunt a subtle bug through a long, noisy history with bisect, including the traps that make bisect lie.

Optional but recommended. The gap between "knows the commands" and "reaches for the right tool under pressure" lives here. Budget ~1–1.5 hours each.
