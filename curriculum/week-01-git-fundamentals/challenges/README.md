# Week 1 — Challenges

Two open-ended problems. Unlike the exercises, these have **no single right answer** — they test whether you can *reason about* the object model, not just run commands.

1. **[Reconstruct the object DB](challenge-01-reconstruct-the-object-db.md)** — given a sequence of commands, predict what the object database and refs look like, then verify.
2. **[Recover a "lost" commit](challenge-02-recover-a-lost-commit.md)** — a commit vanished from the branch. Get it back using `reflog` + the object database.

Optional but recommended. This is where "I know the commands" turns into "I understand Git." Do them after all three lectures and the exercises.

Write up each challenge in a short `writeup.md` — the reasoning matters more than the commands.
