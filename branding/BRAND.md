# C31 · Crunch Version Control — Brand Guide

- **Accent:** Code Crunch blue `#2463EB` (with deep `#1d4ed8`).
- **Voice:** precise, calm, demystifying. Git is a model, not magic — always explain *why* a command does what it does before the command itself.
- **Icon motif:** the commit graph (nodes + edges).
- **One-liner:** "Branch, review, and ship without fear."
